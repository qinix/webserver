# Webserver

A simple, static web server for front-end development.

## Installation

Add this line to your application's Gemfile:

    gem 'webserver'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install webserver

## Usage

    $ webserver -p $PORT -r $DocumentRoot
